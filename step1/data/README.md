# Step 1 Data

이 폴더는 `step1` 실습에서 사용하는 샘플 데이터를 보관합니다.

## 포함된 작은 데이터

| 파일 | 용도 |
| --- | --- |
| `data.csv` | CSV 파일 읽기 예제 |
| `sample.json` | JSON 파일 읽기 예제 |
| `test.txt` | 텍스트 파일 입출력 예제 |
| `hello.txt`, `data.txt`, `backup.txt` | 파일 쓰기/읽기/복사 예제 |
| `users.csv` | CSV 파일 직접 생성/읽기 예제 |
| `student.csv`, `student_100.csv` | Pandas 실습 데이터 |
