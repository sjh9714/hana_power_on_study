# 1. temperature 개념

LLM(OpenAI API 포함)에서 `temperature`는 **출력의 랜덤성(무작위성)** 을 조절하는 값입니다.
쉽게 말하면:

* 낮으면 → **정답형 / 안정적 / 보수적**
* 높으면 → **창의적 / 다양함 / 예측 불가능**

을 의미합니다.

---

LLM은 다음 단어를 확률적으로 선택합니다.

예를 들어:

```text
오늘 날씨가 매우 ______
```

모델 내부 확률:

| 단어       | 확률  |
| ---------- | ----: |
| 좋다       | 70%   |
| 맑다       | 20%   |
| 화창하다   | 8%    |
| 환상적이다 | 2%    |

temperature는 이 확률 분포를 조절합니다.

---

# 2. temperature 값별 특징

| temperature | 특징                   |
| ----------- | ---------------------- |
| 0           | 거의 동일한 답         |
| 0.1 ~ 0.3   | 매우 안정적            |
| 0.5         | 균형                   |
| 0.7 ~ 1.0   | 자연스럽고 다양        |
| 1.2 이상    | 창의적이지만 오류 증가 |

# 5. 내부 동작 원리

temperature는 확률을 다음처럼 조절합니다.

원래 확률:

| 단어 | 확률 |
| ---- | ---: |
| A    | 0.7  |
| B    | 0.2  |
| C    | 0.1  |

---

## temperature 낮음

```python
temperature = 0.1
```

결과:

| 단어 | 변경 후 |
| ---- | ------: |
| A    | 0.95    |
| B    | 0.04    |
| C    | 0.01    |

→ 가장 높은 확률만 거의 선택

---

## temperature 높음

```python
temperature = 1.5
```

결과:

| 단어 | 변경 후 |
| ---- | ------: |
| A    | 0.5     |
| B    | 0.3     |
| C    | 0.2     |

→ 다양한 단어 선택 가능

# 6. OpenAI API 예제

파일명 : step7.py

```python
from openai import OpenAI
import os

client = OpenAI(
    api_key=os.getenv("OPENAI_API_KEY")
)

response = client.chat.completions.create(
    model="gpt-5",
    temperature=0.7,
    messages=[
        {
            "role": "system",
            "content": "너는 친절한 AI 튜터이다."
        },
        {
            "role": "user",
            "content": "Docker란 무엇인가?"
        }
    ]
)

print(response.choices[0].message.content)
```

# 8. temperature와 hallucination 관계

temperature가 높을수록:

* 창의성 증가
* 다양성 증가
* 사실 오류 증가
* hallucination 증가 가능

즉:

```text
높은 temperature = 더 자유로운 상상
```

입니다.

---

# 9. temperature와 top_p 차이

많이 헷갈리는 부분입니다.

## temperature

```text
확률 분포를 얼마나 평평하게 만들지
```

조절

---

## top_p

```text
상위 몇 % 확률 단어만 사용할지
```

조절

예:

```python
top_p=0.9
```

→ 상위 90% 확률 단어만 후보로 사용

# 14. 실험 코드

아래 코드를 실행하면 차이를 쉽게 볼 수 있습니다.

```python
from openai import OpenAI
import os

client = OpenAI(
    api_key=os.getenv("OPENAI_API_KEY")
)

question = "AI란 무엇인가?"

for temp in [0, 0.3, 0.7, 1.2]:

    response = client.chat.completions.create(
        model="gpt-5",
        temperature=temp,
        messages=[
            {
                "role": "user",
                "content": question
            }
        ]
    )

    print("=" * 50)
    print(f"temperature = {temp}")
    print(response.choices[0].message.content)
```
