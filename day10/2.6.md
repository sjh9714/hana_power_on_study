# 16. Gradio와 연결 

```python id="1m8gm6"
from openai import OpenAI
import gradio as gr
import os

client = OpenAI(
    api_key=os.getenv("OPENAI_API_KEY")
)

def chat(message, history):

    response = client.chat.completions.create(
        model="gpt-4.1-mini",
        messages=[
            {
                "role":"user",
                "content":message
            }
        ]
    )

    return response.choices[0].message.content

demo = gr.ChatInterface(chat)

demo.launch()
```

# Gradio 와 FastAPI 연동 방법

Gradio와 FastAPI를 연동하면:

```text id="7g6t3m"
FastAPI = 백엔드 API 서버

```

구조를 만들 수 있습니다.

즉:

```text id="3qpjlwm"
FastAPI 서버 위에
Gradio UI를 붙이는 방식
```

입니다.

# 3. 가장 기본적인 연동 예제

파일명 : step3.py

```python id="wjlwm1a"
from openai import OpenAI
import gradio as gr
import os
from fastapi import FastAPI

client = OpenAI(
    api_key=os.getenv("OPENAI_API_KEY")
)

def chat(message, history):

    response = client.chat.completions.create(
        model="gpt-4.1-mini",
        messages=[
            {
                "role":"user",
                "content":message
            }
        ]
    )

    return response.choices[0].message.content

demo = gr.ChatInterface(chat)

# -----------------------------
# FastAPI 생성
# -----------------------------
app = FastAPI()

# -----------------------------
# FastAPI에 Gradio 연결
# -----------------------------
app = gr.mount_gradio_app(
    app,
    demo,
    path="/"
)

```
