# 1. ChatGRT 유사 프로그램 개발시 가장 많이 사용하는 구조

## FastAPI + 프론트엔드 UI 라이브러리

실무에서는 보통:

* 백엔드:

  * FastAPI
* 프론트엔드:

  * React
  * 또는 Vue.js
* 채팅 UI:

  * React/Vue 채팅 컴포넌트 사용

이 조합이 가장 많습니다.

대표적인 채팅 UI 라이브러리:

| 라이브러리          | 특징                           |
| ------------------- | ------------------------------ |
| react-chat-elements | ChatGPT 비슷한 UI 쉽게 구현    |
| chat-ui-kit-react   | 메시지 버블/입력창 완성도 높음 |
| react-chatbot-kit   | 챗봇 기능 포함                 |
| vue-advanced-chat   | Vue 기반                       |
| gradio              | AI 데모용 매우 쉬움            |
| streamlit-chat      | 빠른 프로토타입                |

장점:

* 몇 줄로 ChatGPT UI 생성
* FastAPI와 통합 가능
* 스트리밍 지원
* Hugging Face 연동 쉬움
* LLM 데모 제작 매우 빠름

예시:

```python
import gradio as gr
from fastapi import FastAPI

app = FastAPI()

def chat(message, history):
    return f"당신이 입력한 내용: {message}"

demo = gr.ChatInterface(chat)

app = gr.mount_gradio_app(app, demo, path="/")
```

실행:

```bash
uvicorn main:app --reload
```

그러면 거의 ChatGPT 비슷한 화면이 바로 생성됩니다.

# 5. FastAPI에서 중요한 핵심 기술

## (1) Streaming Response

ChatGPT처럼 한 글자씩 출력하려면:

```python
from fastapi.responses import StreamingResponse
```

사용합니다.

---

## (2) WebSocket

실시간 양방향 채팅:

```python
from fastapi import WebSocket
```

---

## (3) SSE(Server Sent Events)

LLM 스트리밍에서 많이 사용

```python

```
