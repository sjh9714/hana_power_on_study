# 신용카드 거래 데이터 분석 문제

`credit_card_transactions.csv` 파일을 이용하여 2019년도 VIP를 선정하는 프로그램을 작성합니다 


## 1. VIP 고객 분석

카드번호(cc_num)를 기준으로 다음 값을 계산하시오.

| 계산값    | 설명          |
| --------- | ------------- |
| total_amt | 총 거래금액   |
| avg_amt   | 평균 거래금액 |
| tx_count  | 거래건수      |
 
---

## 2. VIP 점수 계산

다음 공식을 이용하여 VIP 점수를 계산하시오.

$$
VIP\ Score = total_amt \times 0.5 + avg_amt \times 0.3 + tx_count \times 0.2
$$
---

## 3. VIP 고객 분류

* VIP 점수 상위 1%를 VIP 고객으로 선정하시오.
* 나머지는 NORMAL로 분류하시오.

추가 컬럼:

```text
grade
```

---

## 4. 지역별 VIP 고객 수 분석

VIP 고객만 대상으로:

* 지역(state)별 고유 카드 고객 수를 계산하시오.
* 내림차순 정렬하시오.
