# ChatGPT처럼 실시간 스트리밍 출력

OpenAI API는 기본적으로 응답 전체 생성 완료 후 한 번에 반환 방식입니다.

그래서 현재 코드는 ChatGPT가 답변 전체 생성 완료 후 한 번에 화면 출력 됩니다.

목표 : 한 글자씩 또는 토큰 단위로 실시간 출력 하게 구현 하는 방법을 학습합니다 

---

# 핵심 방법

OpenAI API에서:

```python id="jlwm2v"
stream=True
```

를 사용합니다.

그리고 Gradio에서는:

```python id="jlwm3w"
yield 문자열
```

를 사용합니다.

---

## step4.py

```python id="jlwm7a"
from openai import OpenAI
import gradio as gr
import os

client = OpenAI(
    api_key=os.getenv("OPENAI_API_KEY")
)

# --------------------------------
# 스트리밍 채팅 함수
# --------------------------------
def chat(message, history):

    response = client.chat.completions.create(
        model="gpt-4.1-mini",
        messages=[
            {
                "role": "user",
                "content": message
            }
        ],
        stream=True
    )

    partial_message = ""

    # --------------------------------
    # 스트리밍 데이터 수신
    # --------------------------------
    for chunk in response:

        delta = chunk.choices[0].delta.content

        if delta is not None:

            partial_message += delta

            # 실시간 출력
            yield partial_message

# --------------------------------
# Gradio Chat UI
# --------------------------------
demo = gr.ChatInterface(
    fn=chat,
    title="Streaming ChatGPT"
)

# -----------------------------
# FastAPI 생성
# -----------------------------
app = FastAPI()

# -----------------------------
# FastAPI에 Gradio 연결
# -----------------------------
app = gr.mount_gradio_app(
    app,
    demo,
    path="/"
)
```
