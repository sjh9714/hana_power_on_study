# Mac에서 Docker Desktop 없이 순수 Docker Engine 중심으로 사용하는 방법

---

# Mac에서 Docker Engine만 사용하는 현실적인 방법

macOS는 Linux 커널이 아니므로:

```text id="sahfd8"
Docker Engine 직접 실행 ❌
```

이 기본적으로 불가능합니다.

즉:

```text id="jz9slv"
dockerd
```

를 macOS 네이티브에서 직접 실행할 수 없습니다.

왜냐하면 Docker는 Linux 커널 기능:

* cgroups
* namespaces
* overlayfs

를 필요로 하기 때문입니다.

---

# 따라서 가능한 방법

실제로는 아래 방식 중 하나를 사용합니다.

| 방식                   | 추천도 | 특징           |
| ---------------------- | -----  | -------------- |
| Colima + Docker Engine | ★★★★★  | 가장 추천      |
| Lima + Docker          | ★★★★   | 경량 Linux VM  |
| OrbStack               | ★★★★★  | 매우 빠름      |
| Podman                 | ★★★★   | Docker 대체    |
| Docker Desktop         | ★★★    | GUI 포함       |

---

# 가장 추천하는 방식

# → Colima + Docker CLI

이 방식은:

* Docker Desktop 불필요
* 무료
* 가볍고 빠름
* 실무적
* Kubernetes 가능

입니다.

---

# 최종 구조

```text id="rmkb1k"
macOS
 ├─ Homebrew
 ├─ Docker CLI
 └─ Colima
      └─ Linux VM
           └─ Docker Engine
```

---

# 1단계 — Homebrew 설치

설치:

[Homebrew 공식 사이트](https://brew.sh?utm_source=chatgpt.com)

---

## 설치 명령

```bash id="vhjlwm"
/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"
```

확인:

```bash id="bzdu6e"
brew --version
```

---

# 2단계 — Docker CLI 설치

```bash id="8vaxh3"
brew install docker docker-compose
```

확인:

```bash id="u7lr8m"
docker --version
```

---

# 3단계 — Colima 설치

```bash id="kg9a03"
brew install colima
```

확인:

```bash id="92vvsx"
colima version
```

---

# 4단계 — Colima 시작

가장 기본:

```bash id="hqdoip"
colima start
```

자동으로:

* Linux VM 생성
* Docker Engine 실행

됨.

---

# 5단계 — Docker 연결 확인

```bash id="n79b5p"
docker ps
```

정상 동작하면 성공.

---

# 6단계 — 테스트 컨테이너 실행

```bash id="vqjlwm"
docker run hello-world
```

---

# 7단계 — Nginx 테스트

```bash id="c71vq9"
docker run -d -p 8080:80 nginx
```

브라우저:

```text id="f03u9s"
http://localhost:8080
```

---

# 8단계 — Docker Compose 사용

compose.yml:

```yaml id="5b85o6"
services:
  web:
    image: nginx
    ports:
      - "8080:80"
```

실행:

```bash id="wjlwmr"
docker compose up -d
```

---

# 9단계 — 리소스 설정

CPU/메모리 지정 가능.

예:

```bash id="g3mg7t"
colima start --cpu 4 --memory 8
```

# 10단계 — Apple Silicon(M칩) 최적화

