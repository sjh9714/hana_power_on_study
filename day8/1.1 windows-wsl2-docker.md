# 6단계 — Docker 설치 

```bash id="vvyc8d"

#1. Docker의 공식 사이트에 설치 파일 받기 
curl -fsSL https://get.docker.com -o get-docker.sh

#2. Docker 설치
sudo sh get-docker.sh

#3. 도커 실행상태 확인
sudo systemctl status docker

#4.sudo 권한 없이 docker 사용 하기
sudo usermod -aG docker $USER

#5. 로그아웃 하지 않고 docker 명령사용 할 수 있음
newgrp docker 

#6. docket 버전 확인 
docker -v
```

# 7단계 — Docker 자동 시작 설정

WSL2는 systemd 활성화 필요.

---

## systemd 활성화

설정 파일 생성:

```bash id="q0m9k3"
sudo nano /etc/wsl.conf
```

내용:

```ini id="44i5w4"
[boot]
systemd=true
```

저장 후 종료.

---

## WSL 종료

PowerShell:

```powershell id="btowzc"
wsl --shutdown
```

다시 Ubuntu 실행.

---

# 8단계 — Docker 서비스 시작

Ubuntu:

```bash id="rvlxv0"
sudo systemctl start docker
```

자동 시작:

```bash id="y7dx6u"
sudo systemctl enable docker
```

확인:

```bash id="7q6a7u"
systemctl status docker
```

# 9단계 — Docker Compose 테스트

## compose.yml 생성

```bash id="gqk4y4"
mkdir ~/docker-test
cd ~/docker-test
```

```bash id="9s7qsl"
nano compose.yml
```

내용:

```yaml id="duhh5w"

  nginx:
    image: nginx
    ports:
      - "8080:80"
```

실행:

```bash id="m4yzr1"
docker compose up -d
```

브라우저:

```text id="uqm6vv"
http://localhost:8080
```

Nginx 화면 나오면 성공.
